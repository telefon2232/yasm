# YASM — пример на Rust

## Быстрый старт

### 1. Скомпилировать пример с отладочной информацией

```bash
cargo build
```

По умолчанию `cargo build` (debug-профиль) включает DWARF-отладку.
Бинарник будет в `target/debug/yasm-example`.

Для сборки с оптимизациями и отладкой одновременно:

```bash
cargo build --release
```

и поменять путь в `.yasm.json` на `./target/release/yasm-example`.

### 2. Открыть папку rust в VS Code

```bash
code .
```

### 3. Запустить

1. `Ctrl+Shift+P` → **YASM: Show Assembly**
2. Справа откроется панель с дизассемблером
3. Строки исходника и ассемблера окрашены в одинаковые цвета

## Структура

```
rust/
├── Cargo.toml      — манифест проекта
├── src/
│   └── main.rs     — точка входа + функции (square, sum_squares)
├── .yasm.json      — конфиг для YASM
└── README.md       — эта инструкция
```

Rust-пример — однофайловый проект. Функции `square` и `sum_squares`
аналогичны C-примеру, что позволяет сравнить генерируемый ассемблер
между языками.
