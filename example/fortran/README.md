# YASM — пример на Fortran

## Быстрый старт

### 1. Скомпилировать пример с отладочной информацией

```bash
gfortran -g -O1 -o main main.f90 math_utils.f90
```

Флаги:
- `-g` — DWARF-отладка (обязательно, без неё маппинг не работает)
- `-O1` — оптимизации, чтобы ассемблер был интереснее (можно `-O0`, `-O2`)

### 2. Открыть папку fortran в VS Code

```bash
code .
```

### 3. Запустить

1. `Ctrl+Shift+P` → **YASM: Show Assembly**
2. Справа откроется панель с дизассемблером
3. Строки исходника и ассемблера окрашены в одинаковые цвета

## Структура

```
fortran/
├── main.f90        — точка входа
├── math_utils.f90  — модуль с функциями (square, sum_squares)
├── .yasm.json      — конфиг для YASM
└── README.md       — эта инструкция
```

Два файла (`main.f90` и `math_utils.f90`) показывают работу YASM
с многофайловым Fortran-проектом — модуль компилируется вместе с main,
маппинг строк работает для всех исходников.
